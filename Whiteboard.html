Shortcuts & tips

Tools: V select · H pan · P pen · E eraser · R rect · O ellipse · L line · A arrow · T text · S sticky · M mindmap · F flowchart · B bell
Zoom: Scroll wheel to zoom in/out (hold Shift to scroll-pan vertically)
Pan: Hold space + drag, or middle-click drag
Paste: ⌘V pastes text OR images from clipboard onto the canvas
Copy: Select an object and press ⌘C
Delete: Select and press Delete / Backspace
Undo/Redo: ⌘Z / ⌘⇧Z
PPTX: Export your slides to PDF first, then use the 📄 PDF button.
Tabs: Double-click a tab — or hover and click ✎ — to rename. Right-click for a menu (rename / duplicate / close). Tabs autosave to your browser; use Save/Load to back up to a file.
Collapse panels: The ⟨ tab on the left edge hides/shows the tool sidebar. The "Style ▾" button hides/shows the colour & size bar.
Random picker: 🎲 Random picks a number from a range, or a name from a pasted list. Tick "Don't repeat" to make sure no one gets called twice in a row.
Text: Select text, then use the colour swatches or Size slider to change it. Drag a corner handle to resize selected text or shapes.
Connectors: Select an existing mindmap/flow node, then create a new node to auto-connect it. Or use Connector and click one node, then another node; arrows snap to the nearest edge.
